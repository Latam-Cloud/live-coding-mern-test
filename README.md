## Purpose
This repository is a starter boilerplate for a live coding interview.

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB (can be mocked)

## Instructions
- You do NOT need to add authentication
- You may mock MongoDB or explain schema logic
- Focus on clean code and reasoning, not setup

## How to Run
### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm start
